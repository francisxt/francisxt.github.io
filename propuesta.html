1. Introducción

FG Tech se complace en presentar esta propuesta actualizada para el diseño, desarrollo e implementación de un Sistema Integral de Gestión de Parqueadero y Car Wash para el Holiday Inn Airport.

Nuestra solución consiste en una plataforma web centralizada y tres aplicaciones móviles para Android. Estas herramientas están diseñadas para optimizar las operaciones diarias, digitalizar el flujo de trabajo y mejorar la eficiencia, el control y la experiencia del cliente.

2. Alcance del Proyecto

El proyecto contempla el desarrollo y entrega de los siguientes componentes:

  • Plataforma Web de Administración Central: Panel de control para gestión de roles, tarifas, supervisión y reportería.
  • Tres (3) Aplicaciones Móviles para Android:
    • App "Garita": Para el registro inicial de vehículos.
    • App "Seguridad": Para la gestión de ingreso, asignación y salida.
    • App "Lavado": Para el control del proceso de limpieza.

Nota: Esta propuesta contempla el desarrollo para Android con distribución interna. El desarrollo para iOS o la publicación en tiendas de aplicaciones no está incluido en este alcance.

3. Descripción de Módulos

Plataforma Web de Administración

Será el centro de control del sistema, accesible desde cualquier navegador web. Sus funcionalidades clave incluyen:

  • Dashboard Principal: Visualización en tiempo real de la ocupación, vehículos en lavado y estadísticas de ingresos.
  • Gestión de Parqueadero: Mapa visual con el estado de cada espacio (ocupado, disponible, limpio, sucio).
  • Gestión de Usuarios y Roles: Creación de cuentas para personal con permisos definidos (Administrador, Gerencia, Operación).
  • Módulo de Tarifas y Facturación: Panel para configurar tarifas por tipo de estancia y servicio de lavado.
  • Módulo de Reportería: Generación de informes exportables (PDF, Excel) para Ama de Llaves, Seguridad y Contabilidad.

App: Garita

  • Registro Rápido: Formulario optimizado para capturar placa, marca, color y datos del cliente.
  • Ticket Digital: Al registrar, envía automáticamente un ticket con código QR al correo del cliente.
  • Sincronización Instantánea: La información se refleja de inmediato en la App de Seguridad.

App: Seguridad
