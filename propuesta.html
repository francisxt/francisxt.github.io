1. Introducción

FG Tech se complace en presentar esta propuesta actualizada para el diseño, desarrollo e implementación de un Sistema Integral de Gestión de Parqueadero y Car Wash para el Holiday Inn Airport.

Nuestra solución consiste en una plataforma web centralizada para la administración y la reportería, junto con tres aplicaciones móviles intuitivas para la plataforma Android. Estas herramientas están diseñadas para optimizar las operaciones diarias de su personal en garita, seguridad y lavado de vehículos. El objetivo es digitalizar y automatizar el flujo de trabajo, desde el ingreso del vehículo hasta su salida y limpieza, mejorando la eficiencia, el control y la experiencia del cliente.

Será el centro de control del sistema, accesible desde cualquier navegador web.

    • [cite_start]

      [cite_start]

4. Cronograma de Desarrollo

El proyecto se ejecutará en un marco de tiempo estimado de 2 semanas, distribuido de la siguiente manera:

  • Semana 1: Backend, Base de Datos y Plataforma Web
    • Días 1-2: Arquitectura del sistema, diseño de base de datos y configuración del módulo de tarifas.
    • Días 3-5: Desarrollo del núcleo de la plataforma web, gestión de roles y sistema de reportería.
  • Semana 2: Aplicaciones Móviles, Integración y Despliegue
    • Días 6-8: Desarrollo de las tres aplicaciones móviles para Android.
    • Días 9-10: Integración final, pruebas de campo, despliegue en la infraestructura del cliente y capacitación.
